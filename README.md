# InstruNet-AI 🎵

A CNN-based music instrument recognition system using audio signals converted
into spectrogram images for deep learning classification.

---

## 📁 Dataset & Project Files

Due to GitHub file size limitations, the complete dataset and generated
spectrogram images are hosted on Google Drive.

🔗 **Google Drive Link (Shared for all collaborators):**  
https://drive.google.com/drive/folders/1_chZ4Q46__2pXOpz_I7jLL8cg4vTf0gp

### Drive Folder Contains:
- Raw audio files (.wav)
- Spectrogram images generated from audio
- Preprocessed datasets (.npy / .npz)
- Training and validation data

---

## 🚀 How to Use
1. Open the Google Drive link above
2. Download the required folders
3. Place them inside the project directory
4. Run the CNN training / evaluation scripts
